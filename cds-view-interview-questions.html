CDS View Interview Questions and Answers

A focused guide to SAP CDS View interview topics including associations, annotations, access control, performance and virtual data models.

What is a CDS View?

A CDS View is a semantic data model defined in ABAP development tools and used heavily in S/4HANA applications.

What are CDS associations?

Associations define relationships between CDS entities and allow path expressions for related data access.

What are annotations?

Annotations add metadata for UI, analytics, authorisation, semantics and service behaviour.

How do you optimise CDS performance?

Use proper joins, avoid unnecessary fields, understand cardinality, push filters down and validate SQL plans where needed.
