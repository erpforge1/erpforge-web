ABAP Interview Questions

1. What is SAP ABAP?

ABAP is SAP’s programming language used to build reports, enhancements, interfaces, forms and business logic in SAP systems.

2. What is the difference between transparent, pooled and cluster tables?

Transparent tables map directly to database tables. Pooled and cluster tables are older storage concepts and are not preferred for modern S/4HANA development.

3. What are internal tables?

Internal tables are runtime data structures used to store multiple records in ABAP programs.

4. Difference between standard, sorted and hashed internal tables?

Standard tables are index-based, sorted tables maintain key order and hashed tables provide key-based access optimised for exact lookups.

5. What is a work area?

A work area holds one row structure used to read, modify or append data to an internal table.

6. What is a field symbol?

A field symbol is a reference-like placeholder that points to data objects dynamically and can avoid unnecessary copying.

7. What is the difference between SELECT SINGLE and UP TO 1 ROWS?

SELECT SINGLE is intended for key-based single-row access. UP TO 1 ROWS is commonly used when the first row based on conditions or ordering is needed.

8. What is FOR ALL ENTRIES?

FOR ALL ENTRIES is used to select related records based on values in an internal table. The driving table should not be empty.

9. How do you improve ABAP performance?

Reduce database calls, select required fields only, use proper keys, avoid nested loops on large data and push logic to the database where appropriate.

10. What is modularisation in ABAP?

It means dividing logic into reusable units such as methods, function modules, forms and classes.

11. What is Object-Oriented ABAP?

OO ABAP uses classes, objects, methods, inheritance, interfaces and encapsulation to build maintainable applications.

12. What is an enhancement?

An enhancement allows custom functionality to be added without modifying SAP standard objects directly.

13. What is a BADI?

Business Add-In is an object-oriented enhancement technique used to implement custom logic at predefined points.

14. What is a user exit?

A user exit is an older enhancement technique that allows customer-specific logic in SAP standard processes.

15. What is ALV?

ABAP List Viewer is used to display structured report output with sorting, filtering, layout and export options.

16. What is a CDS View?

Core Data Services views define semantic data models on top of database tables and are central to S/4HANA development.

17. What is RAP?

RESTful ABAP Programming Model is SAP’s modern programming model for building transactional Fiori-ready services.

18. What is OData?

OData is a protocol used to expose and consume data services, commonly used between SAP backend and Fiori/UI applications.

19. What is debugging in ABAP?

Debugging allows developers to inspect runtime behaviour, variable values, call stacks and logic flow.

20. What is SAT?

SAT is a runtime analysis tool used to measure ABAP performance and find expensive processing blocks.
